Instrument & Watchlist Console

Upload 100k+ instrument dumps, run fuzzy searches, and monitor live prices via Kite Connect.

1. Authenticate

A default admin user is available for demos:
admin@example.com / admin123

2. Configure Kite Credentials

Provide your Kite Connect API key and the daily access token. Tokens are stored in-memory for the current server session. Use the test button to verify connectivity.

3. Upload Instrument CSV

The importer accepts CSV files with the following header columns: instrument_token, exchange_token, tradingsymbol, name, last_price, expiry, strike, tick_size, lot_size, instrument_type, segment, exchange.
